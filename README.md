# LANG 美語與日語自學系統

一份離線可開、單檔 HTML 的語言自學工具，第一版聚焦 **美語 + 日語**，並把聽力放在核心流程。

## 怎麼用

1. 直接打開 `index.html`
2. 選擇美語或日語課程
3. 用「聽力實驗室」播放、慢速播放、聽寫、跟讀自評
4. 進度與錯句存在瀏覽器 localStorage

## 第一版範圍

- 美語：生活會話、旅行/點餐、連音與美式發音重點
- 日語：問候自我介紹、請求/確認、店家與交通常用句
- 聽力：瀏覽器 Web Speech API 播放，支援正常/慢速/循環
- 練習：聽寫檢查、跟讀自評、完成標記、錯句收藏
- 匯出：可複製 Markdown 學習紀錄

## 注意

- 語音播放依賴瀏覽器內建語音。Chrome/Edge 通常可用；可用聲音會因電腦而異。
- 本系統是學習工具，不取代真人老師或正式語言測驗教材。
- 單檔 HTML，無 build step、無套件安裝。
